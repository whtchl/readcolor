# readcolor

ReadColor.java  列出一张照中所有像素的rgb.

ImageFindDemo.java : keyImagePath:标准图片，对比图片

                     screenShotImage:拍照的照片
                     
                     keyImagePath的长宽小于screenShotImage的大小
                     
                     keyImagePath的像素和screenShotImage中某个方块的像素一样的话就显示匹配成功。
